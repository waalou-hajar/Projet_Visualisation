Objectif Principal

Dataviz est la version raccourcie de data visualisation. Comme son nom l’indique, il s’agit d’outils pour mettre en forme de manière lisible un ensemble de données brutes.
Le but de notre projet est de résumer et visualiser l’information pertinente contenue dans un jeu de données. Pour la mise en pratique, nous allons utiliser un outil très puissant appelé Tableau.
Tableau permet de créer des tableaux, des graphiques, des infographies, des cartes à partir de n’importe quelles données (SQL, Hadoop, Excel…). L’interface est particulièrement intuitive et permet de transférer les données par simple glisser-déposer. Les graphiques sont mis à jour en temps réel et partageables.

Problématique :

Dans notre projet nous allons étudier l’impact de la pandémie COVID-19 sur l’économie mondiale de chaque pays.

Démarche

À l’aide de Tableau nous allons créer plusieurs graphiques en visualisant différentes informations contenues dans notre modèle qui examine l'impact potentiel de COVID-19 sur l'économie des 182 pays. Par la suite nous allons utiliser des tableaux de bord représentant un ensemble de plusieurs vues, permettant de comparer simultanément une variété de données. Par exemple, si nous disposons d'un ensemble de vues que nous examinons tous les jours, nous pouvons créer un tableau de bord qui affiche toutes les vues à la fois, plutôt que de naviguer vers des feuilles de calcul distinctes.

Data Set

Data Set est sur Kggle cliquez Ici pour le voir .
Ce modèle examine l’impact de la pandémie COVID-19 sur l’économie mondiale de chaque pays.

Les attributs

  • CODE : les codes alphabétiques sur 3 caractères pour les noms de pays.
  • COUNTRY : les pays.
  • DATE : la date représentée par YY-MM-DD.
  • HDI (Human Development Index) : L’indice de développement humain, c’est un critère pour évaluer le développement d'un pays.
  • TC (Total Cases) : les cas confirmés.
  • TD (Total Deaths) : les cas décès.
  • STI (Stringency Index) : l’indice de stringence représente les politiques de confinement (les fermetures d'écoles et de lieux de travail, les événements publics, les transports en commun), les politiques de maintien à la maison.
  • POP : la population.
  • GDPCAP (Gdp Per Capita) : le PIB par habitant
